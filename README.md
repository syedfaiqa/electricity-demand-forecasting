#  Electricity Demand Forecasting – UI Dashboard

A frontend UI dashboard for electricity demand forecasting that visualizes analytics,
factor impact, and trends using interactive charts.

##  Features
- Electricity demand analytics dashboard
- Factor impact visualization (temperature, holidays, season, etc.)
- Interactive charts and graphs
- Clean and responsive UI

##  Tech Stack
- HTML
- CSS
- JavaScript
- Chart.js

##  Project Structure
- templates/ → UI HTML pages
- app.py → Backend connector (if applicable)
- prediction13.csv → Sample dataset
- models/ → Supporting files

##  How to Run
Open the HTML files directly in a browser  
or use Live Server extension in VS Code.
